# widgets
Widgets is a centralized repository for storing reusable widgets developed for the GoHighLevel (GHL) platform. It serves as a shared library to manage, maintain, and organize all custom widgets created for various GHL use cases.

WA - whatsapp link generator
bar - bar chart for leads by hour
donut - pv project + source AND project + user
funnel - funnel 
report - leads aging report (based on op) AND report widget (contain user and pipeline and how many opportunities under that user)
twilio - webhook
